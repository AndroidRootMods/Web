How to update WhatsApp Aero?


Advertisements - Reklamlar


❗️ WARNING: ❗️
Do not uninstall the WhatsApp Aero app that is currently installed on your device to update. Install the new update directly on it.

Step 1:
You will see an icon style selection screen as in the screenshot below:


You can choose between "Modern Icons" or "Classic Icons" from the icon selection screen. There is no feature difference between "Modern Icons" or "Classic Icons", only some icons have been changed.

Click here if you want to see the differences between "Modern Icons" or "Classic Icons" with screenshots.

Step 2:
After choosing between Modern or Classic icons, you will see a package name selection screen as in the screenshot below:


Package name selection is important. If you choose the wrong package name, WhatsApp Aero will be installed from scratch and will ask you to verify the number again, so you must choose the correct package name.

To choose the correct package name, check what is written under the "Package" text in Aero Privileges > About shown step by step in the screenshot below, so you will know the package name you are currently using.


Check your own package name as shown in step 3 of the screenshot above.


⚪ If your package name is com.aero, select the com.aero package from the package selection screen.

⚪ If your package name is com.whatsapp, select the com.wa package from the package selection screen.

⚪ If your package name is com.aerolla, select the com.aerolla package from the package selection screen.

Step 3:

After selecting your package name, tap one of the download buttons as seen in the screenshot below.

(There are multiple download buttons so that you can download WhatsApp Aero from multiple alternative sites in case you run into problems. If you have problems with any download buttons, try the other download buttons.)


When you do everything right, you will have updated WhatsApp Aero.
