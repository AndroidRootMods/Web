Which package name should I install?

Let's briefly tell you.

First of all, we have to say that we are recommending you to install com.wa package.

The com.aero and com.aerolla packages are developed as an alternative for people who have WhatsApp installed on their device by default and do not remove.

You must uninstall the original WhatsApp application from your device to install the com.wa package name. You have the opportunity to use many applications (such as sticker apps and similar applications) developed for WhatsApp along with the com.wa package name.

It is almost impossible to use applications such as sticker applications in com.aero and com.aerolla package names.


Already using WhatsApp Aero?
Then check what it says under "Package" in Aero Privileges > About.
