  • I can't see anyone's online/last seen info.

    Disable "Freeze Last Seen" feature in Khater Privileges ≫ Privacy.

  • How can I hide that I am online?

    Enable "Freeze Last Seen" feature in Khater Privileges ≫ Privacy.

    Remember; when you enable this feature, you will not be able to see other WhatsApp users information such as "online", "last seen", "typing...", "recording...".

  • I am having "Checking for new messages" issue and my messages are late/not going.

    When you encounter a connection checking error and your messages are delivered late or you receive messages late, it's usually an issue with your use of privacy features.

    On the Khater side, the only factor affecting message speed is your use of Khater's privacy features. Apart from the privacy features of Khater, there is no Khater factor that affects the speed.

    ☀️ Solution proposals:
    1-) Disable the features listed below:

    🔸 Khater Privileges ≫ Privacy ≫ Custom Privacy (Delete all contacts in Custom Privacy page).

    🔸 Khater Privileges ≫ Privacy ≫ Show Blue Ticks after reply.

    🔸 Khater Privileges ≫ Privacy ≫ Contacts (Disable all options in Contacts).

    🔸 Khater Privileges ≫ Privacy ≫ Groups (Disable all options in Groups).

    🔸 Khater Privileges ≫ Privacy ≫ Broadcasts (Disable all options in Broadcasts).

    After disabling all, go to the main screen and restart Khater a few times using the "Restart WhatsApp" option from the options. If your message speed problem is caused by Khater, turning off privacy features of Khater and restarting Khater will fix the problem.
    ‏‏‏‏‏‏‏‏

    2-) Disable airplane mode feature of Khater. Click here to learn how to disable airplane mode.
    ‏‏‏‏‏‏‏‏

    3-) Check your internet connection and WhatsApp server status. The problem may be due to your internet connection or WhatsApp being temporarily unavailable.

    • Backup and Restore

  • How to backup messages in WhatsApp Khater?

    Firstly, make sure you have given the following permissions to WhatsApp Khater:

    1-) Permission to access storage (necessary).
    2-) Permission to install from external/unknown sources (not mandatory but recommended).


    After making sure you have the necessary permissions:
    🔸 Start the backup process with the Khater Privileges ≫ Universal ≫ Backup and restore ≫ "Backup WhatsApp data" option.
    Your backups are transferred to WhatsApp ≫ aeroYedek folder.

    🔸 Then take a second backup with own settings of WhatsApp ≫ Chats ≫ "Chat backup" option.

    ⚠️ WARNING ⚠️
    During the backup process, please do not close the app or put it in the background, wait until the backup process is finished.

  • How to restore backups in WhatsApp Khater?

    Firstly, make sure you have given the following permissions to WhatsApp Khater:

    1-) Permission to access storage (necessary).
    2-) Permission to install from external/unknown sources (not mandatory but recommended).


    To restore your backups you need to make sure your backups are in the correct folder. To verify that your backups are in the correct folder:

    1-) Go to Khater Privileges ≫ About and check what your package name is (it could be "com.aero", "com.whatsapp" or "com.aerolla").
    2-) Now that you learned your package name, use your file manager to check the name of the folder in the WhatsApp ≫ aeroYedek folder. E.g; if your package name is com.aero, the path to the folder where your backup is located should be WhatsApp ≫ aeroYedek ≫ com.aero. Edit the name of your folder according to the package name you learned in step 1.

    After making sure that your backup is in the correct folder, you can restore your backups with Khater Privileges ≫ Universal ≫ Backup and restore ≫ "Restore WhatsApp data".


    ⚠️ If you are trying to restore your backups and you get an error saying that no backup was found in your storage, it means that your backups is not in the correct folder. Please make sure your backup is in the correct folder.

  • How to change package name without losing messages?

    ⚠️ The following guide is prepared for transition from com.aero package name to com.whatsapp (com.wa for short) package name. However, this guide is valid for every package name, the only thing that changes is the names.(E.g; if you want to change from com.aero package name to com.aerolla package name, you can apply the com.whatsapp articles in the following guide as if it were com.aerolla)

    We assume that you have taken a backup by following the guide under the How to backup messages in WhatsApp Khater? heading.


    1-) After you have your backup, rename the "WhatsApp" folder using your file manager (E.g; rename it to "WhatsApt").
    2-) Download WhatsApp Khater with package name com.whatsapp (com.wa for short) from https://whatsaero.com.
    3-) Install the new WhatsApp Khater you downloaded, open it and close the app without verifying your number at the phone number verification screen.
    4-) Rename the folder you changed in step 1 to "WhatsApp" using your file manager. (If you get a warning that there is already a WhatsApp folder, you can delete or rename the folder named "WhatsApp", then try renaming the folder you changed in step 1 to "WhatsApp").
    5-) Using your file manager, change the name of the folder named "com.aero" in the WhatsApp ≫ aeroYedek folder to "com.whatsapp".
    6-) Reopen the new WhatsApp Khater you installed and continue by verifying your number. After verifying your number, WhatsApp app will ask that you have a backup and if you want to restore, you can restore your backups by tapping "Restore" button or you can restore your backups with Khater Privileges ≫ Universal ≫ Backup and restore ≫ "Restore WhatsApp data" option.


    After you have successfully restored your backups by following these steps, you will have switched from com.aero package name to com.whatsapp package name. You can remove the old WhatsApp Khater with package name com.aero on your device.

  • How can I import my messages from Official WhatsApp to WhatsApp Khater?

    Please follow the steps below in order:

    1-) Make a backup using Official WhatsApp Settings ≫ Chats ≫ "Chat backup" button. Wait until the backup is complete, then close WhatsApp when backup complete.

    2-) Find the folder named "WhatsApp" in your storage using your file manager and rename this folder to any other name, for example: "WhatsApt"

    3-) Download and install WhatsApp Khater on your device from https://whatsaero.com and open the app.

    📌 When you opened the app, if app asks you for permission to access the storage, give permission to access the storage and go to the number verification screen, but close the app without typing your number..

    📌 When you opened the app, If app doesn't ask for storage permission, use your devices settings to allow WhatsApp Khater to access the storage, then reopen WhatsApp Khater and go to the number verification screen, but close the app without typing your number..

    4-) Rename the folder you changed in step 2. to "WhatsApp" using your file manager.

    For example, we changed the name of our folder to "WhatsApt" in step 2., now we will rename this folder to "WhatsApp".

    ⚠️ If you encounter a warning like "This folder already exists" while naming, delete the "WhatsApp" folder using your file manager and try renaming the folder you changed in step 2. to "WhatsApp" using your file manager.

    5-) Reopen WhatsApp Khater, verify your number and continue. If you see a question about Google Drive backup after verifying your number, select "Continue". Then WhatsApp will see your backups and ask if you want to restore, you can restore your messages using the "Restore" button at the bottom left.
